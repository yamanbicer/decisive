## Evidence Standard

Distinguish a *claim* from *evidence*. Default to skepticism; let evidence move you.

1. **Source quality** — Primary (a fetched page, a metric, a trace) beats secondary
   beats assertion. A number with a URL beats an adjective.
2. **Falsifiability** — Would the claim survive one unscripted follow-up? If a result
   only works on a pre-designed input, it's a demo, not a proof.
3. **Base rate** — Compare against the category bar (retention, conversion, accuracy),
   not against zero. "Up 3x" from a tiny base is noise.
4. **Disconfirmation** — Actively search for the counter-evidence (use a tool to find
   competitors or contradicting data) before accepting the happy-path story.
5. **Causation vs correlation** — Did the orchestration/feature CHANGE the outcome, or
   merely accompany it? Demand a delta, ideally a side-by-side.

## How to apply
Name the single weakest-supported claim in the debate, try to refute it with a tool,
and only concede when real evidence clears this bar. Hold the cap until it does.
